# Roadmap: Cyber Security Mastery
Subject: #study/cybersecurity 
Status: #status/active 

## 🗺️ Learning Path & Domains

### Phase 1: 01 - Fundamentals
- [ ] **Core Principles**: CIA Triad, DAD Triad, AAA Framework.
- [ ] **Risk Management**: Vulnerability, Threat, Risk, Risk Assessment (Quantitative vs. Qualitative).
- [ ] **Security Controls**: Administrative, Technical, and Physical controls.
- [ ] **Operating Systems**: Linux hardening, Windows security features.

### Phase 2: 02 - Network Security
- [ ] **TCP/IP Stack**: Deep dive into protocols (TCP, UDP, ICMP, DNS, HTTP/S).
- [ ] **Defensive Devices**: Firewalls (Stateful vs. Next-Gen), IDS/IPS, WAFs, Honeypots.
- [ ] **Network Infrastructure**: VPNs, SD-WAN, Zero Trust Network Access (ZTNA).

### Phase 3: 03 - Web Security
- [ ] **OWASP Top 10**: Injection, Broken Authentication, Cross-Site Scripting (XSS), etc.
- [ ] **APIs & Cloud Apps**: Securing REST/GraphQL APIs, OAuth 2.0 and OpenID Connect.
- [ ] **Tools**: Burp Suite, OWASP ZAP, Nikto.

### Phase 4: 04 - Cryptography
- [ ] **Symmetric vs. Asymmetric**: AES, RSA, ECC.
- [ ] **Hashing & Digital Signatures**: SHA-256, HMAC, PGP/GPG.
- [ ] **PKI (Public Key Infrastructure)**: Certificates, CAs, and CRLs.

### Phase 5: 05 - Endpoint & Malware Analysis
- [ ] **Malware Types**: Ransomware, Trojans, Rootkits, Spyware.
- [ ] **Endpoint Security**: EDR (Endpoint Detection & Response), Antivirus, XDR.
- [ ] **Malware Analysis**: Static vs. Dynamic analysis (Sandbox testing).

### Phase 6: 06 - IAM & Authentication
- [ ] **Identity & Access Management**: Active Directory, LDAP, Kerberos.
- [ ] **Multi-Factor Authentication (MFA)**: TOTP, Biometrics, Hardware keys (YubiKey).

### Phase 7: 07 - Penetration Testing (Offensive Security)
- [ ] **Reconnaissance**: OSINT, Nmap, Shodan, Google Dorking.
- [ ] **Exploitation**: Metasploit, SQL Injection, Buffer Overflows (Basics).
- [ ] **Post-Exploitation**: Privilege Escalation, Lateral Movement, Persistence.

### Phase 8: 08 - Incident Response & Forensics
- [ ] **SIEM (Security Information & Event Management)**: Splunk, ELK, Sentinel.
- [ ] **Digital Forensics**: Disk imaging, Memory forensics, Log analysis.

### Phase 9: 09 - Governance & Compliance
- [ ] **Frameworks**: NIST CSF, ISO 27001, SOC2.
- [ ] **Regulations**: GDPR, HIPAA, PCI-DSS.
