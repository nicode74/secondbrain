# Study Note: Observability with Prometheus and Grafana
Subject: #study/devops/monitoring
Status: #study/learning 

## 📖 Key Concepts & Definitions
- **Prometheus**: A monitoring system and time series database.
- **Grafana**: A multi-platform open-source analytics and interactive visualization web application.
- **Exporters**: Tools that export metrics from third-party systems as Prometheus metrics.

## 📝 Detailed Notes
- Prometheus uses a pull model to collect metrics.
- Grafana connects to Prometheus as a data source to create dashboards.

## 🧠 Questions & Flashcards
- **Q**: What is PromQL?
  - **A**: Prometheus Query Language, used to select and aggregate time series data in real-time.

## ✍️ Practice / Application
- Set up a dashboard to monitor CPU usage of a server.
