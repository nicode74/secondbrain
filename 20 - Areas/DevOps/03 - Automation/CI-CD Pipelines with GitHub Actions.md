# Study Note: CI-CD Pipelines with GitHub Actions
Subject: #study/devops/automation
Status: #study/learning 

## 📖 Key Concepts & Definitions
- **CI (Continuous Integration)**: Automatically building and testing code changes.
- **CD (Continuous Delivery/Deployment)**: Automatically deploying code to production or staging.
- **GitHub Actions**: An automation platform that allows you to create workflows triggered by GitHub events.

## 📝 Detailed Notes
- Workflows are defined in YAML files in `.github/workflows`.
- Use "Runners" to execute jobs.

## 🧠 Questions & Flashcards
- **Q**: What is a "Secret" in GitHub Actions?
  - **A**: Encrypted environment variables used to store sensitive data like API keys.

## ✍️ Practice / Application
- Create a workflow that runs unit tests on every push.
