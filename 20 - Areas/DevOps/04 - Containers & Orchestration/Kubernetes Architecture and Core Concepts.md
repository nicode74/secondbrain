# Study Note: Kubernetes Architecture and Core Concepts
Subject: #study/devops/containers
Status: #study/learning 

## 📖 Key Concepts & Definitions
- **Kubernetes (K8s)**: An open-source system for automating deployment, scaling, and management of containerized applications.
- **Pod**: The smallest deployable units of computing that you can create and manage in Kubernetes.
- **Control Plane**: Manages the worker nodes and the Pods in the cluster.

## 📝 Detailed Notes
- Components: etcd, API Server, Scheduler, Controller Manager.
- Worker Nodes: Kubelet, Kube-proxy, Container Runtime.

## 🧠 Questions & Flashcards
- **Q**: What is a "Service" in Kubernetes?
  - **A**: An abstract way to expose an application running on a set of Pods as a network service.

## ✍️ Practice / Application
- Deploy a multi-node cluster using Minikube.
