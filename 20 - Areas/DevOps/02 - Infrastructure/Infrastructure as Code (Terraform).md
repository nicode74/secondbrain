# Study Note: Infrastructure as Code (Terraform)
Subject: #study/devops/infrastructure
Status: #study/learning 

## 📖 Key Concepts & Definitions
- **IaC (Infrastructure as Code)**: Managing and provisioning infrastructure through machine-readable definition files.
- **Terraform**: An open-source IaC tool by HashiCorp for building, changing, and versioning infrastructure safely.
- **Provider**: Plugins that allow Terraform to interact with cloud providers, SaaS providers, and other APIs.

## 📝 Detailed Notes
- Terraform uses HCL (HashiCorp Configuration Language).
- State management is crucial; it keeps track of the resources created.

## 🧠 Questions & Flashcards
- **Q**: What is the purpose of `terraform plan`?
  - **A**: It shows what actions Terraform will take to reach the desired state specified in the configuration.

## ✍️ Practice / Application
- Provision a simple EC2 instance on AWS.
