# Roadmap: Kotlin & Android Native Professional
Subject: #study/mobile-dev #study/kotlin #study/android 
Status: #status/active 

## 🤖 Native Android Learning Path

### Phase 1: Kotlin (The Language)
1. **01 - Kotlin Fundamentals**
	- [ ] Variables (`val` vs `var`), Data Types, String Templates.
	- [ ] Functions, Default/Named Arguments, Lambda Expressions.
	- [ ] Control Flow: `When`, `If/Else`, Ranges, Loops.
2. **02 - Kotlin OOP & Advanced**
	- [ ] Classes, Constructors, Init blocks, Properties.
	- [ ] Inheritance, Interfaces, Abstract Classes.
	- [ ] **Data Classes**, **Sealed Classes**, **Enum Classes**.
	- [ ] **Coroutines**: Structured concurrency, Dispatchers, Scope.

### Phase 2: Android Studio & UI Fundamentals
3. **03 - Android Studio & UI Basics**
	- [ ] Project Structure, Gradle, Manifest, Resources.
	- [ ] Activity Lifecycle (`onCreate` to `onDestroy`).
	- [ ] Fragments & Navigation Component.
4. **04 - Jetpack Compose Mastery (Modern UI)**
	- [ ] Layouts: `Column`, `Row`, `Box`, `Scaffold`.
	- [ ] State: `remember`, `mutableStateOf`, **State Hoisting**.
	- [ ] Lists: `LazyColumn`, `LazyRow`.
	- [ ] Navigation in Compose.

### Phase 3: Architecture & Data
5. **05 - Architecture & State Management**
	- [ ] **MVVM Pattern**: View, ViewModel, Model.
	- [ ] `LiveData` vs **StateFlow/SharedFlow**.
	- [ ] Dependency Injection with **Hilt** or **Koin**.
6. **06 - Networking & Local Data**
	- [ ] REST APIs with **Retrofit** & **Moshi/Gson**.
	- [ ] Local Database with **Room Persistence Library**.
	- [ ] DataStore (Modern replacement for SharedPreferences).

### Phase 4: Reliability & Production
7. **07 - Background Tasks**
	- [ ] WorkManager for persistent background work.
	- [ ] Push Notifications (FCM).
8. **08 - Testing & Deployment**
	- [ ] Unit Testing (JUnit, Mockk).
	- [ ] UI Testing (Espresso, Compose Testing).
	- [ ] CI/CD with GitHub Actions & Play Store Publishing.

## 🏆 Project Milestones
- [ ] **Kotlin Playbox**: A collection of small Kotlin scripts.
- [ ] **Task Master**: A local task manager app using Room and Compose.
- [ ] **News Hub**: A networking app using Retrofit and MVVM.
