# Bootcamp Roadmap: Dart & Flutter Professional
Subject: #study/mobile-dev #study/dart #study/flutter 
Status: #status/active 

## 🚀 Module-Based Learning Path

### Phase 1: The Dart Language (The Foundation)
1. **01 - Dart Fundamentals**
	- [ ] Variables, Constants (`final` vs `const`), Data Types.
	- [ ] Functions, Optional/Named Parameters, Arrow Syntax.
	- [ ] Collections: Lists, Sets, Maps & The `Spread` Operator.
	- [ ] Control Flow: If/Else, Switch-Case, For/While Loops.
2. **02 - Dart Advanced (OOP & Async)**
	- [ ] Classes & Constructors (Named, Factory, Redirecting).
	- [ ] Inheritance, Mixins, Abstract Classes, Interfaces.
	- [ ] **Null Safety**: `?`, `!`, `late`, `required`.
	- [ ] Asynchronous Programming: `Future`, `async/await`, `Stream`.

### Phase 2: Flutter UI & Core Concepts
3. **03 - Flutter Essentials**
	- [ ] Widget Tree & Element Tree.
	- [ ] Stateless vs Stateful Widgets (Lifecycle: `initState`, `build`, `dispose`).
	- [ ] Material Design vs Cupertino.
4. **04 - Layout & UI Mastery**
	- [ ] Rows, Columns, Stacks, Containers.
	- [ ] Responsive UI: `MediaQuery`, `LayoutBuilder`, `Flexible`, `Expanded`.
	- [ ] Navigation: Navigator 1.0 vs 2.0 (Router API), GoRouter.

### Phase 3: Logic, State & Backend
5. **05 - State Management Deep Dive**
	- [ ] Lifting State Up vs `setState`.
	- [ ] **Provider** & **Riverpod** (Recommended for beginners/pros).
	- [ ] **Bloc** & **Cubit** (For enterprise apps).
6. **06 - Networking & APIs**
	- [ ] JSON Serialization/Deserialization.
	- [ ] `http` & `dio` packages.
	- [ ] Error Handling & Interceptors.
7. **07 - Local Persistence**
	- [ ] `Shared Preferences` for small data.
	- [ ] `SQLite` (via `sqflite` or `drift`) for relational data.
	- [ ] `Hive` for NoSQL.

### Phase 4: Production Readiness
8. **08 - Testing & Deployment**
	- [ ] Unit Testing (Dart).
	- [ ] Widget Testing & Integration Testing.
	- [ ] Flutter DevTools: Debugging & Performance Profiling.
	- [ ] App Store & Play Store Deployment (Fastlane).

## 🏆 Projects to Build
- [ ] **Portfolio App**: Simple static UI.
- [ ] **Weather App**: API integration & JSON handling.
- [ ] **E-commerce App**: Advanced State Management & Auth.
